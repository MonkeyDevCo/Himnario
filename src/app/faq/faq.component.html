FAQ

How to search a song? ¿Cómo buscar una canción?

First, select from home a section (Anthems or Choruses)

Second, Write into searchbar what song do you want to find using a title extract or a number

Finally, select a song from list

Primero, selecciona una sección (Coritos o Himnos)

Luego, escribe dentro del buscador cuál canción deseas buscar usando un extracto del título o un número

Finalmente, selecciona la canción desde el listado

How to change size? ¿Cómo cambiar el tamaño del contenido de una canción?

From home section change song size

Desde la ventana principal, desliza el selector de tamaño de letra hasta escoger la que más te acomode
